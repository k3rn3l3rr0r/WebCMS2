WebCMS2
=======

Content management system based on Nette Framework with Doctrine2 ORM library.

Special features
--------

* Auto updating system + modules.
* Cloning languages.

Contains libraries
------------------

* Nette framework
* Doctrine2
* Nella/doctrine
* Nella/console
* Nella/gedmo
* Doctrine/migrations
* o5/grido
* kdyby/bootstrap-form-renderer

INSTALLATION
------------

Add this line into your composer.json file.

```
"webcms2\webcms2" : "@dev"
```
